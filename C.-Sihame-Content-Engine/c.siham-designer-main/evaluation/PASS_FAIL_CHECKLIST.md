# PASS / FAIL CHECKLIST

## Pass if all are true
- the image matches the post meaning, not just its mood
- Arabic text feels readable and intentionally placed
- the visual mode fits the post type
- symbolism clarifies rather than decorates
- the design feels bespoke rather than template-like
- the signature line, when used, stays secondary

## Fail if any are true
- beautiful but semantically vague
- Arabic hierarchy feels cramped or weak
- wrong visual mode for the post type
- symbolism is random or generic
- output looks like a stylish but irrelevant AI poster
