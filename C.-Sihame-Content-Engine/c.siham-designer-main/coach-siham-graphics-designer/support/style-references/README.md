# Style Reference Library

## Purpose
Store curated reference images that demonstrate the desired typography materiality for Coach Sihame visuals.

## Usage
When Nano Banana (or any image generation model) supports image-to-image style reference, upload one of these images alongside the text prompt to anchor the AI's "Visual DNA."

## What to include
- Examples of Arabic calligraphy with gold leaf diacritics on textured paper
- Examples of clean geometric Arabic sans-serif body text
- Examples of elegant English cursive signatures
- Examples of the overall "ink on cotton paper" aesthetic

## Naming convention
- `calligraphy_gold_diacritics_01.png`
- `geometric_body_text_01.png`
- `signature_cursive_01.png`
- `paper_texture_reference_01.png`
