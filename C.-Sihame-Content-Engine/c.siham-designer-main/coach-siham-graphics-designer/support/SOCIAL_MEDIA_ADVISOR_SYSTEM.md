# SOCIAL MEDIA ADVISOR SYSTEM

## Purpose
Create a small advisor structure around the graphics designer system so feedback can come from distinct expert lenses.

## Advisor lanes

### 1. Design Advisor
Focus:
- composition
- hierarchy
- luxury vs generic feel
- negative space
- premium restraint

### 2. Typography Advisor
Focus:
- Arabic readability
- line height
- title/body relationship
- text density
- text placement

### 3. Social Content Advisor
Focus:
- whether the visual fits the post function on Instagram/Facebook/Telegram
- clarity vs over-design
- whether the design helps the post perform

### 4. Meaning Alignment Advisor
Focus:
- does the visual express the meaning, not just the mood?
- are the symbols relevant?
- does the design support the emotional/educational role of the post?

## How to use them
Every serious visual test should be reviewed through these 4 lenses.

## Output format for any advisor
- strongest thing
- weakest thing
- top 2 fixes
- approve / iterate / reject

## Rule
Do not ask advisors vague questions.
Always give:
- the post text
- the chosen visual mode
- the generated image
- the intended meaning
