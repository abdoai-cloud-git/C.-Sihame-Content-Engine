# TYPOGRAPHY RULES

## Purpose
Define typography behavior for Coach Sihame visuals so Arabic readability and premium tone are protected.

## Core principle
Typography is not decoration.
It is the primary carrier of meaning in many Coach Sihame visuals.

---

## Typography feeling
The typography should feel:
- calm
- elegant
- grounded
- premium
- emotionally safe
- editorial, not flashy

It must never feel:
- corporate
- childish
- trendy for trend's sake
- wedding-invitation decorative
- compressed or noisy

---

## Arabic-first rules
### 1. Readability is non-negotiable
- Never shrink Arabic text to force too much content into one frame
- If text is too long, convert to carousel or editorial reflection layout
- Prioritize comfortable line-height
- Give Arabic generous breathing room

### 2. Hierarchy must be obvious
Use clear distinction between:
- headline / hook
- sub-line
- body text
- CTA / action line

### 3. Spacing matters
- wide margins
- generous paragraph spacing
- avoid cramped boxes
- avoid text touching the visual edge

### 4. Contrast must support softness
- enough contrast to read easily
- avoid harsh black-on-neon or ultra-hard visual strain
- premium softness > low-contrast weakness

---

## Layout by post type
### Quote card
- short phrase only
- one dominant text block
- large negative space
- no decorative quote marks unless highly restrained

### Reflection visual
- editorial paragraph structure
- body-led hierarchy
- no oversized headline unless necessary
- line-height must be noticeably generous

### Educational post
- strongest hierarchy required
- headline > segmented points > conclusion
- use dividers or spacing, not clutter

### Promo / offer visual
- highest clarity
- offer title first
- key detail second
- CTA third
- elegant but unmistakable reading order

---

## Font consistency rule
### The Dual-Font System
- The Coach Sihame brand embraces a **Dual-Font System** to bridge spiritual elegance and supreme readability.
- Typography must be described to the AI as **physical material on paper**, not as abstract font names.

### Accent title rule (The Calligraphic Headline)
- **Title / Headline:** Use an **Elegant Farsi-style Arabic Calligraphy (خط الفارسي)** for the main headline.
- **Materiality descriptors (use these in prompts):**
  - "Flowing Farsi-style calligraphy with a gentle diagonal lean"
  - "Fluid, connected letterforms with elegant thin-to-thick stroke variation"
  - "Drawn with a fine pointed nib, warm champagne gold ink"
  - "Gold leaf accents on diacritics and tashkeel marks" (champagne/muted gold, not bright metallic)
- Style reference: خط الفارسي — use this as a *human reference*, not in prompts.
- This creates visual warmth, emotional flow, and a premium handcrafted feel.
- The calligraphic font acts as the artistic anchor for the post.

### Support / Body text rule
- **Support / Body Text:** Use a **clean, modern geometric Arabic typeface** as the baseline for all body and support text.
- **Materiality descriptors (use these in prompts):**
  - "Clean, thick, corporate-style geometric letterforms"
  - "Sharp, precise sans-serif strokes with even weight distribution"
  - "Modern Kufic-inspired geometric clarity"
- Human reference: Rubik — but do not use the name "Rubik" in image generation prompts.

### Signature rule
- **Signature:** The name "Siham Atamnia" should always be rendered in an **elegant, light, cursive English handwriting script**.
- **Materiality descriptors (use these in prompts):**
  - "Delicate pen-written cursive, as if signed by hand with a fine-point ink pen"
  - "Light ink impression, elegant and restrained"

### Paper materiality rule
- All typography should be described as **physical ink rendered on textured paper**.
- Prompt-level anchor: "Typography rendered as real calligraphy ink on soft, textured cotton paper with subtle grain."
- This materiality bridge is what makes AI-generated text look premium instead of digitally flat.

## Typography do nots
- no bubbly/funky fonts
- no overly condensed Arabic styles
- no too-many-font combinations (stick strictly to Calligraphic Title + Rubik Body + Script Signature)
- no Canva-template-looking quote card typography
- no unstable font switching across posts without a reason

---

## Practical rule
If the design must choose between:
- more visual flair
- or better Arabic readability

Always choose:
## better Arabic readability
