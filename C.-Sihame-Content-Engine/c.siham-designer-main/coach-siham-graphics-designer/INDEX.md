# Coach Siham Graphics Designer — Index

## Purpose
This folder contains the evolving knowledge system for Coach Siham’s graphics designer brain.

The goal is to create a visual intelligence engine that turns post meaning into brand-consistent visuals using:
- brand logic
- symbolism
- typography
- visual modes
- prompt construction
- evaluation

## Working principle
The system should think in this order:
1. brand meaning
2. emotional territory
3. post function
4. visual mode
5. symbols
6. titling
7. typography (including explicit font roles: title/body/signature)
8. prompt
9. evaluation

## Current known weakness
The system already matches **mood** reasonably well.
The main ongoing problem is improving **meaning alignment** so visuals become less generic and more semantically correct for each post.

## Recommended next phase
Build a small benchmark set of posts and evaluate each generated visual using the meaning alignment scorecard.
