# SYMBOLIC VISUAL LANGUAGE

## Purpose
This file defines the symbolic visual language for Coach Sihame’s graphics designer brain.

It exists so the system can create visuals that explain emotional, somatic, and psychological ideas through **simple, premium, symbolic forms** instead of relying only on:
- generic backgrounds
- stock photos
- quote cards
- cliché spiritual imagery

The goal is to make symbolism feel:
- elegant
- minimal
- psychologically intelligent
- emotionally safe
- spiritually grounded
- highly readable in Arabic layouts

---

## Core Principle
Symbols are not decoration.
They are **meaning containers**.

Each symbol should help the viewer understand:
- a state
- a transition
- a contrast
- a mechanism
- a relationship

If a symbol does not clarify meaning, it should not be used.

---

## Symbol Categories

## 1. Nervous System / State Symbols
Use these when the post is about regulation, activation, overwhelm, shutdown, or inner safety.

### Allowed symbolic forms
- smooth circles
- soft concentric rings
- flowing wave lines
- tangled vs untangled line forms
- expanding vs contracted shapes
- contained forms inside open space
- grounded horizontal lines
- soft pulse/ripple motifs

### Meanings
- **smooth circle** = regulation / wholeness / containment
- **concentric rings** = layers / depth / inner system / resonance
- **gentle ripple** = nervous system response / subtle impact / regulation spreading
- **tangled line** = overwhelm / fragmentation / mental-emotional entanglement
- **untangled line** = clarity / integration / settling
- **contracted shape** = fear / guarding / constriction
- **open expanded shape** = safety / breath / capacity / receiving

---

## 2. Contrast Symbols
Use these when the post compares two inner states or two concepts.

### Good comparison themes
- mindful vs mind-full
- regulation vs calming
- boundary vs wall
- reaction vs response
- surface behavior vs root cause
- contraction vs expansion
- fragmentation vs integration
- people-pleasing vs self-honoring

### Allowed symbolic logic
- split composition
- two-column contrast
- mirrored forms with different tension
- clean before/after contrast
- one chaotic symbol vs one integrated symbol

### Rule
The contrast must be obvious within 1–2 seconds.
The viewer should understand:
> “these two states are different”
without needing a long explanation first.

---

## 3. Inner Parts / Maestro Symbols
Use for content about inner multiplicity, leadership of the self, inner children, or internal conflict.

### Allowed symbolic forms
- small circles orbiting a larger calm center
- multiple threads converging into one conductor point
- scattered shapes gathered into one gentle structure
- a central anchor with surrounding movement
- a soft container holding multiple inner elements

### Meaning logic
- **center point** = self / inner leadership / maestro
- **orbiting parts** = inner selves / internal states / parts in movement
- **convergence** = integration / listening / leadership / internal coordination
- **containment shape** = safe holding / internal organization

### Avoid
- literal orchestra illustrations
- cartoon children
- therapy-worksheet aesthetics

---

## 4. Spiritual / Reflective Symbols
Use for reflective, prayerful, or deeply contemplative posts.

### Allowed symbolic forms
- soft light beam
- horizon line
- still water
- subtle opening / doorway shape
- delicate radiance
- soft empty space
- one grounded stone / one anchored object (used minimally)

### Meaning logic
- **light** = guidance / clarity / opening / witnessing
- **horizon** = perspective / surrender / spaciousness
- **still water** = calm / emotional settling / receptivity
- **opening shape** = permission / transition / spiritual expansion

### Avoid
- lotus clichés
- glowing third eyes
- over-mystical cosmic patterns
- fairy dust / magical sparkle effects
- mandalas unless extremely restrained and deeply integrated

---

## 5. Educational Diagram Symbols
Use when the post needs explanation more than atmosphere.

### Allowed symbolic forms
- circles with labels
- layered stacks
- root/branch structures
- pathway diagrams
- soft containers
- simple dividers
- minimal arrows only when necessary
- step-based visual sequencing

### Mood
- clear
- grounded
- elegant
- emotionally intelligent
- never cold/corporate

### Avoid
- startup SaaS icons
- stock infographic look
- aggressive arrows everywhere
- generic brain icons / clipart

---

## Symbol Style Rules

### Must be
- minimal
- smooth
- soft-edged
- refined
- clear at small scale
- compatible with premium typography
- supportive of text, not competing with it

### Must not be
- loud
- cartoonish
- clipart-like
- flat generic UI icons
- overly decorative
- hyper-detailed
- spiritually kitsch

---

## Composition Rules
When using symbols:

1. **One main symbolic idea per visual**
   - do not combine too many metaphors at once

2. **Text remains the main teacher when needed**
   - symbol supports meaning; it does not replace clarity

3. **Negative space matters**
   - symbols must leave breathable room for Arabic text

4. **Symbol tension must match emotional tension**
   - if the message is soft, symbol cannot feel aggressive
   - if the message is educational, symbol cannot feel vague

5. **No random symbolism**
   - if a shape is used, the reason should be explainable

---

## Good Use Cases
This symbolic language is strongest for posts about:
- nervous system states
- emotional regulation
- inner parts / maestro
- people-pleasing dynamics
- boundaries vs walls
- overwhelm vs safety
- root vs surface healing
- stillness, surrender, awareness, witnessing

---

## Bad Use Cases
This symbolic language is weaker when the post requires:
- direct offer communication
- very practical logistics
- explicit event details
- heavy text density without room for symbolic support

In those cases, use:
- educational structure
- promotion/offer clarity
- text-first layout
instead of forcing symbolism.

---

## Prompting Guidance
When symbolic visuals are selected, prompts should include:
- one central metaphor
- emotional tone
- composition direction
- negative space requirement
- soft premium palette
- explicit simplicity

### Example prompt fragments
- "a single untangled line emerging from a dense knot"
- "concentric rings dissolving softly into open space"
- "a calm center point with scattered forms gathered around it"
- "a split composition showing contraction on one side and openness on the other"
- "a gentle ripple expanding through still water with large negative space"

---

## Final Rule
A symbolic visual for Coach Sihame should feel like:
- a concept made visible
- a feeling made clear
- a subtle emotional teaching tool

It should never feel like:
- a decorative moodboard
- a generic wellness poster
- a random abstract illustration with no conceptual discipline
