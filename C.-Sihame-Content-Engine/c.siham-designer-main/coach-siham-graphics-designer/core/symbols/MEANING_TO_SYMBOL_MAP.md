# MEANING TO SYMBOL MAP

## Purpose
Map post meaning into simple symbolic cues that remain elegant, premium, and non-cliché.

## Rule
Symbols should be:
- simple
- readable
- meaning-led
- restrained

Not decorative for the sake of decoration.

---

## Growth / flourishing / expansion
### Meanings
- growth
- flourishing
- inner expansion
- becoming
- soft development

### Allowed symbols
- delicate stem
- small unfolding leaf
- subtle green shoot
- branching line with restraint
- expanding rings

### Use carefully
- muted green, soft botanical cues, minimal composition

### Avoid
- loud flower fields
- generic wellness floral backgrounds
- decorative bouquet aesthetics

---

## Safety / containment
### Meanings
- regulation
- holding
- grounded calm
- self-soothing

### Allowed symbols
- nested forms
- soft enclosing curve
- centered held shape
- gentle hand-like abstract form
- warm contained light

---

## Chaos / overwhelm
### Meanings
- fragmentation
- nervous system overload
- mental scatter

### Allowed symbols
- tangled line
- scattered points
- compressed shape
- visual noise on one side only

### Rule
Must be contrasted against a calmer state if used.

---

## Clarity / integration
### Meanings
- understanding
- settling
- integration
- inner coherence

### Allowed symbols
- untangled line
- open ring
- aligned pathway
- coherent center

---

## Reflection / stillness
### Meanings
- pause
- witnessing
- contemplation
- surrender

### Allowed symbols
- soft light beam
- horizon line
- still water
- empty calm space
- subtle texture

---

## Spiritual groundedness
### Meanings
- divine closeness
- depth
- presence
- inner spaciousness

### Allowed symbols
- light, but subtle
- opening shape
- horizon / sky-wash
- still reflective surface

### Avoid
- lotus cliché
- glowing mystical kitsch
- fairy dust / cosmic overload

---

## Educational contrast
### Meanings
- one concept vs another
- surface vs root
- boundary vs wall
- mindful vs mind-full

### Allowed symbols
- split composition
- two-state comparison
- tangled vs clear
- closed vs open
- dense vs breathable

---

## Example mappings
- الازدهار / النمو → subtle green shoot, unfolding leaf, expanding ring
- الأمان → held center, nested curve, warm contained light
- التهدئة → softer outer atmosphere
- التنظيم → coherent center, aligned flow, open ring
- الفوضى → knot, scatter, dense scribble
- الوعي → light touch, opening shape, widening space

## Final rule
If a symbol can be replaced by a calmer simpler one, choose the simpler one.
