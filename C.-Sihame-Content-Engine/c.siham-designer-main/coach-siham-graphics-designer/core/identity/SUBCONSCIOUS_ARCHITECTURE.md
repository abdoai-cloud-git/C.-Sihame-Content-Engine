# SUBCONSCIOUS ARCHITECTURE

## Purpose
Define the internal hierarchy of the Coach Siham graphics designer brain.

The system must not think like a prompt generator.
It must think like a layered subconscious that translates brand meaning into visual decisions.

---

## Core rule
Higher layers override lower layers.

If a lower layer proposes something that conflicts with a higher layer, the higher layer wins.

---

## Layer hierarchy

### Layer 1 — Brand Core (highest priority)
Files:
- `BRAND_CORE_SHEET.md`
- `BRAND_SIGNAL_EXTRACTION.md`

Role:
- define who the brand is
- define what the brand is not
- define emotional territory
- define core metaphors
- define the promise and the meaning of flourishing / stillness / safety

This layer answers:
- What world are we in?
- What must never be violated?

---

### Layer 2 — Visual Identity
Files:
- `VISUAL_IDENTITY_RULES.md`
- `TYPOGRAPHY_RULES.md`
- `DESIGN_DO_NOTS.md`

Role:
- define taste
- define palette behavior
- define readability standards
- define visual restraint
- define what feels premium vs generic

This layer answers:
- What does this brand look and feel like?

---

### Layer 3 — Meaning System
Files:
- `SYMBOLIC_VISUAL_LANGUAGE.md`
- `MEANING_TO_SYMBOL_MAP.md`
- `CONCEPT_CONTRAST_VISUALS.md`
- `GUIDED_PRACTICE_VISUALS.md`

Role:
- translate meaning into symbols, contrasts, and embodied cues
- prevent mood-only image choices

This layer answers:
- What visual logic best expresses this post’s meaning?

---

### Layer 4 — Post-Type Logic
Files:
- `VISUAL_DECISION_MAP.md`
- `VISUAL_TYPES.md`
- `VISUAL_MODES_FRAMEWORK.md`
- `POST_TO_DESIGN_TEXT_MAP.md`

Role:
- classify the post
- choose the correct visual mode
- choose text-first vs symbol-first vs atmosphere-first

This layer answers:
- What kind of visual does this post need?

---

### Layer 5 — Text Expression
Files:
- `TITLING_LAYER.md`
- `TYPOGRAPHIC_LAYOUT_ENGINE.md`

Role:
- decide whether title is needed
- decide what text belongs on-image vs caption
- shape Arabic reading hierarchy

This layer answers:
- How should the message be expressed on the design?

---

### Layer 6 — Prompt Construction
Files:
- `PROMPT_PATTERNS.md`
- `NANOBANANA_OPERATIONAL_RULES.md`
- `LEGACY_SKILL_MERGE.md`

Role:
- convert design decisions into executable image prompts
- define iteration discipline

This layer answers:
- How do we instruct the model without losing meaning or brand fidelity?

---

### Layer 7 — Evaluation / Correction
Files:
- `MEANING_ALIGNMENT_SCORECARD.md`
- `VISUAL_EVALUATION_WORKFLOW.md`
- `SOCIAL_MEDIA_ADVISOR_SYSTEM.md`

Role:
- judge output quality
- find the top 2 problems
- drive the next iteration

This layer answers:
- Is the output good enough? If not, what specifically must change?

---

## Golden rule
A visual should pass through these questions in order:
1. Is it faithful to the brand?
2. Does it express the post meaning?
3. Does it fit the correct visual mode?
4. Does it support Arabic text properly?
5. Can it be executed cleanly by the image model?
6. Does it pass meaning-alignment review?

If the answer is no at any stage, the design must be corrected before approval.
