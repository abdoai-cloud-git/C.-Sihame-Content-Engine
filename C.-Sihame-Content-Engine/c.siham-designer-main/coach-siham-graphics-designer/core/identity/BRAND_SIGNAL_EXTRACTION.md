# BRAND SIGNAL EXTRACTION

## Purpose
Convert the brand core sheet into operational signals the graphics designer brain can actually use.

## 1. Primary visual promise
The design must communicate:
- أمان قبل التغيير
- السكون الذي يقود إلى الازدهار
- عمق بلا قسوة
- وقار روحي بلا وعظ ثقيل

## 2. Emotional territory to preserve
Always bias toward:
- الأمان
- السكون
- الرحمة
- الوضوح
- الاتساع
- الوصل
- الاحتواء
- الحضور
- الهداية

Avoid visual emotional territory of:
- الضغط
- العجلة
- الإثارة الصاخبة
- الصدمة التسويقية
- العدوانية

## 3. Core visual meanings
- **Teal deep** → regulation, quiet trust, depth
- **Sage green** → healing, slow growth, fitrah, flourishing
- **Warm ivory** → breathing room, softness, purity
- **Soft muted gold** → light, divine trace, sacred refinement
- **Warm stone grey** → root, maturity, silent containment

## 4. Symbol priorities
Prefer symbols tied to:
- nervous system settling
- containment
- inner leadership / maestro
- growth from inside outward
- river vs stagnation
- lamp/light through glass
- movement from pain to meaning

## 5. Typography meaning
Typography should feel like:
- a session, not a slogan
- guidance, not advertising
- grounded breath, not pressure

## 6. Visual modes that fit this brand best
### Strongest modes
1. Typography-first reflection
2. Symbolic/conceptual educational
3. Embodied/atmospheric guided-practice

### Weakest modes
- loud promotional poster
- generic stock lifestyle aesthetic
- overly mystical spiritual collage
- trendy influencer wellness look

## 7. Growth symbol rule
If using growth symbols like green, leaves, or flowers:
- keep them subtle
- make them meaning-led
- use them for flourishing / fitrah / expansion only
- never let them become decorative botanical filler

## 8. Quick design test
A visual is probably right if it feels:
- calm but not dead
- spiritual but not cliché
- feminine but not ornamental
- premium but not fake luxury
- safe but still clear
