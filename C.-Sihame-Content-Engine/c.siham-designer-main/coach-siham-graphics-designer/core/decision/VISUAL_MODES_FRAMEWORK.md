# VISUAL MODES FRAMEWORK

## Purpose
Force the designer brain to choose the right mode before generating visuals.

## Mode 1 — Typography-first
Use when:
- the words are the main carrier of value
- reflection post
- prayer/reflection
- long-form thoughtful text

### Goal
Give Arabic text the strongest readability and emotional space.

---

## Mode 2 — Symbolic / conceptual
Use when:
- the post explains an inner mechanism
- there is a contrast between two states
- symbolism can clarify meaning better than a scene

### Goal
Make the concept visible.

---

## Mode 3 — Embodied / atmospheric
Use when:
- the post is somatic
- the post guides regulation or self-contact
- the feeling in the body matters more than abstract explanation

### Goal
Create felt safety, containment, and embodied presence.

---

## Mandatory decision questions
Before generating any prompt, the designer brain must answer:
1. What is the **meaning cue**?
2. What is the **mood cue**?
3. Which visual mode is correct?
4. Does this need text-first, symbol-first, or atmosphere-first treatment?

## Anti-drift rule
If the system cannot justify the chosen mode in one sentence, the direction is too vague and should be rejected.
