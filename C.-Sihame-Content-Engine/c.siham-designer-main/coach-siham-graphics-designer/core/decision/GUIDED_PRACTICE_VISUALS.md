# GUIDED PRACTICE VISUALS

## Purpose
Define how visuals should work for guided-practice posts.

These posts are not just reflective. They are instructional, embodied, and regulating.

## Core rule
A guided-practice visual must communicate:
- safety
- containment
- self-contact
- slowness
- embodied reassurance

Not just generic calmness.

## Best visual direction
Use one of these:
1. **Embodied symbolic**
   - subtle cue of hands on chest / abdomen
   - non-literal, non-stock-photo
2. **Containment metaphor**
   - nested soft forms
   - centered holding shape
   - gathered warmth
3. **Text-first with somatic support**
   - strong Arabic text block
   - calm support image behind it

## Must feel
- intimate
- safe
- grounded
- premium
- gently instructive

## Must avoid
- generic linen-only moodboards
- random soft backgrounds with no body meaning
- over-literal meditation stock photos
- wellness cliché poses

## Prompt rule
Every guided-practice image prompt must include:
- one embodiment cue
- one safety/containment cue
- Arabic-friendly text space
- emotional softness

## Review gate
Reject if the image is:
- only pretty
- not embodied
- not clearly related to the practice
