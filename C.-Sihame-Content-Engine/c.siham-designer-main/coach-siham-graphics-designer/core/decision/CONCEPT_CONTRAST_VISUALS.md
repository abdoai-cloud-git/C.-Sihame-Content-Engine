# CONCEPT CONTRAST VISUALS

## Purpose
Define how to visualize two contrasting inner states or concepts.

Use for posts like:
- regulation vs calming
- boundary vs wall
- mindful vs mind-full
- surface vs root
- contraction vs expansion

## Core rule
The contrast should be understood in 1–2 seconds.

## Best structures
1. **Split composition**
   - left/right contrast
2. **Before / after**
   - tense vs integrated
3. **Tangled vs clear**
   - same motif in two states
4. **Open vs closed**
   - same form with different tension

## Good symbol pairs
- knot vs untangled line
- contracted circle vs open ring
- dense scribble vs calm orbit
- hard wall block vs breathable boundary line
- scattered points vs coherent center

## Must feel
- clear
- elegant
- educational
- emotionally intelligent
- premium, not cheap infographic

## Must avoid
- startup-style diagrams
- clipart icons
- childish comparison graphics
- loud arrows and labels everywhere

## Prompt rule
Every concept-contrast prompt must include:
- two distinct states
- one shared symbolic logic
- clean negative space
- restrained palette

## Review gate
Reject if:
- contrast is not obvious
- symbolism feels random
- layout feels generic infographic
