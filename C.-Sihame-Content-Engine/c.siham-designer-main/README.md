# c.siham designer

A sandbox repo for fixing, restructuring, and strengthening the Coach Siham designer system as a real visual intelligence engine.

## Purpose
This repository exists to improve the designer system without disturbing the main working copy in the broader workspace.

The aim is not just to generate pretty visuals.
The aim is to build a system that:
- reads post meaning correctly
- chooses the right visual mode
- protects Arabic readability
- stays emotionally safe and brand-consistent
- produces outputs that feel bespoke rather than generic

## Repo structure
- `coach-siham-graphics-designer/` — system files
  - `core/` — source-of-truth design logic
  - `support/` — supporting docs and advisory layers
  - `references/` — operational and reference guidance
  - `scripts/` — helper scripts
- `examples/` — example structured outputs
- `outputs/` — generated visual artifacts kept for reference
- `SOURCE_OF_TRUTH.md` — defines what governs the system
- `WORKFLOW.md` — canonical end-to-end workflow
- `MULTITASK_PLAN.md` — active execution plan
- `TODO.md` — immediate priorities
- `ROADMAP.md` — strategic direction
