## القسم الثالث: أفعال تحبها سهام مقابل أفعال ترفضها

### ✅ الأفعال الحية (تستخدمها سهام)
| الفعل | السياق الصحيح |
|---|---|
| **ذاب / يذوب** | "القبض الذي في الصدر ذاب" |
| **توسّع / اتّسع** | "جهاز عصبي يتسع لهذا التحرر" |
| **فُكّت** | "العقد التي بداخلك فُكّت" |
| **احتوى** | "يحتوي الألم بدلاً من مقاومته" |
| **رويداً رويدا** | إيقاع التهدئة التدريجية |
| **يتدحرج** | "كرة الثلج المتدحرجة" |
| **يُنضح / يسري** | "دع هذه الحالة تنضح في النبرة" |
| **يُرخي** | "أرخي قبضتك" |
| **يستقبل** | "هل نحن مستعدون لاستقبال نسخنا الأوسع؟" |
| **يتجدد** | "الاستقبال يعني في عمقه التجدد كل يوم" |

### ❌ الأفعال المرفوضة (لا تستخدمها سهام أبداً)
| الفعل | سبب الرفض |
|---|---|
| **اقتلع / اجتث** | لغة عنف، لا لغة تشافٍ |
| **حارب / تغلّب** | تُمثّل الصراع الداخلي كحرب |
| **كسر** | تكسير الحواجز ليست لغتها |
| **ادفع / اصرع** | لغة القوة التجارية المبالغ فيها |
| **تجاوز** | يوحي بقفز على المشاعر لا عبورها |
| **تخلّص من** | تُعامل الأجزاء الداخلية كنفايات |
| **اقطع / ابتعد** | توصية قطعية بلا عمق سياقي |

## القسم الرابع: إيقاع التنفس البصري

سهام تكتب كما يتنفس الجسد. هذه قواعد التنفس البصري:

```
كل فكرة = سطر
كل إحساس = سطر وحده
الصمت بين السطرين = نفَس
```

**مثال صحيح:**
```
استيقظت صباحاً

وكأنني عدت من رحلة طويلة

جسدي كان يتذكر كل محطة

الأنفاس — هدأت
```

**مثال خاطئ:**
```
استيقظت صباحاً وأحسست أن جسدي يتذكر كل محطة من الرحلة الداخلية وكأنني عدت من مكان بعيد وأنفاسي بدأت تهدأ تدريجياً.
```

- `#سهام_عثامنية` دائماً في النهاية

- دعوة مبيعات مباشرة وضاغطة
