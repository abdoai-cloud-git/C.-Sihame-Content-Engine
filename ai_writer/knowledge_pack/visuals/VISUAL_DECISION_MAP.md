# VISUAL DECISION MAP

This document outlines the logic the visual decision engine uses to map the input text (caption/post idea) to the appropriate visual output.

## Logic Engine

When analyzing a post, the brain must follow this sequential evaluation:

### 1. The Reflective/Spiritual Path
*   **Trigger**: The text contains deep introspection, poetry, spiritual journaling, surrendering, connecting with the Divine, or quiet psychological realization.
*   **Decision**: 
    *   **Atmosphere**: Softer, atmospheric, ethereal.
    *   **Space**: Massive negative space.
    *   **Structure**: Less rigid, more fluid.
    *   **Tone**: Emotional elegance.
    *   **Visual Elements**: Symbolic imagery (light, water, quiet landscapes), or entirely text-first with a sublime background texture.
    *   **Type Example**: Quote Card, Reflection Visual, Symbolic Emotional Visual.

### 2. The Educational Path
*   **Trigger**: The text teaches a concept, breaks down a psychological mechanism (e.g., trauma response, attachment styles), offers a framework, or provides step-by-step guidance.
*   **Decision**:
    *   **Atmosphere**: Clear, focused, well-lit.
    *   **Hierarchy**: Strong headline, clear subheaders, distinct segmentation.
    *   **Structure**: Highly structured layout (grids, lists, subtle dividing lines).
    *   **Tone**: Grounded, authoritative but accessible.
    *   **Visual Elements**: Infographic-style layouts, clean typography, subtle diagrams.
    *   **Type Example**: Educational Post, Infographic-Style Post, Carousel Cover.

### 3. The Promotional Path
*   **Trigger**: The text announces an offer, a retreat, a masterclass, coaching slots, or an event.
*   **Decision**:
    *   **Atmosphere**: Confident, bright, highly legible.
    *   **Hierarchy**: Offer name > Dates/Details > Clear Call-to-Action (CTA).
    *   **Structure**: Direct alignment (center or strong left-aligned). Less abstract.
    *   **Tone**: Invitational, elegant, premium.
    *   **Visual Elements**: High-quality lifestyle imagery, clear bold typography, elegant buttons or arrows. Never aggressive or cluttered.
    *   **Type Example**: Promotion / Offer Visual.

### 4. The Story/Intimate Path
*   **Trigger**: The text is a personal anecdote, a behind-the-scenes thought, a quick realization, or a direct, conversational message to the audience.
*   **Decision**:
    *   **Atmosphere**: Warm, human, spontaneous but curated.
    *   **Space**: Can be more cropped or close-up.
    *   **Structure**: Soft, less polished/rigid than grid posts.
    *   **Tone**: Subtle intimacy, connection.
    *   **Visual Elements**: Warm lighting, personal elements (coffee cup, notebook, soft portrait, moving scenes if video), simple typography placed considerately.
    *   **Type Example**: Story Visual.
